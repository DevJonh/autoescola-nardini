# Breakpoints

Breakpoints for responsive layout:

<table>
  <tr>
    <th>Extra Small</th>
    <td>Default</td>
  </tr>
  <tr>
    <th>Small (sm)</th>
    <td> >={BreakPoints.sm}px</td>
  </tr>
  <tr>
    <th>Medium (md)</th>
    <td> >={BreakPoints.md}px</td>
  </tr>
  <tr>
    <th>Large (lg)</th>
    <td> >={BreakPoints.lg}px</td>
  </tr>
  <tr>
    <th>Extra Large (xl)</th>
    <td> >={BreakPoints.lg}px</td>
  </tr>
</table>

Using in styled component:

```js
import { breakAt, BreakPoints } from "./BreakPoints";

const Title = styled.h1`
  font-size: 1rem;
  ${breakAt(BreakPoints.md)} {
    font-size: 2rem;
  }
`;
```
